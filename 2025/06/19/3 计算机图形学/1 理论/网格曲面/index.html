面

使用索引表表示面。例如有顶点表。

0 -> (0, 0, 0)  // v0
1 -> (1, 0, 0)  // v1
2 -> (0, 1, 0)  // v2

Triangle1: (0, 1, 2)  // 由 v0, v1, v2 组成

一个面一般由一个数组表示。

细分曲线

Chaikin细分

每条边的 1/4 和 3/4 处插入两点，并且相连

细分曲面

拓扑规则：插入新的点线面

几何规则：计算新旧顶点位置

由初始网格经过一定的拓扑规则和几何规则得到新的网格

优点：

  • C1 or C2 连续
    • C¹ 连续 (一阶连续性或切线连续性)：指的是曲面上任意一点的切线方向是连续变化的。在视觉上，这意味着曲面没有尖锐的棱角，连接处是光滑过渡的。
    • C² 连续 (二阶连续性或曲率连续性)：指的是曲面上任意一点的曲率是连续变化的。这比 C¹ 连续的要求更高，保证了曲面不仅光滑，而且其弯曲的程度也是平滑变化的。这样的曲面在视觉上会显得更加完美和自然，光线照射在上面时，高光部分的过渡会非常平滑，没有突变。
  • 可以表示任意拓扑
  • 适合动画建模，快速渲染
  • 部分细分曲面有解析表达

缺点：

  • 奇异点处没有解析表达，难以计算微分量
    • 奇异点：四边形网格中邻接边数不等于4，三角网格中邻接边数不等于6
  • 不容易构造高阶连续曲面
