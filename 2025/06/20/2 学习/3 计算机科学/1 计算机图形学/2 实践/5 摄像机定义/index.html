摄像机定义

摄像机作为组建，添加 Camera 类，当前文件结构如下

app_3d_camera
├── Application.cpp
├── Application.h
├── main.cpp
├── components
│   ├── Camera.cpp
│   └── Camera.h
└── scene
    ├── World.cpp
    └── World.h

位置参数

  • 位置 X Y Z

方向参数

  • 表示方向的三个正交基向量，以及欧拉角
  • 世界上坐标，用来更新 Right，否则摄像机将找不到方向。同时方便上下移动。

属性

  • 移动速度
  • 视角移动速度（鼠标灵敏度）
  • 视场角
#ifndef CAMERA_H
#define CAMERA_H

#include "GL/glew.h"
#include "glm/glm.hpp"
#include "glm/gtc/matrix_transform.hpp"

// 定义几种可能的摄像机移动方式，作为ProcessKeyboard函数的输入
enum Camera_Movement {
    FORWARD,
    BACKWARD,
    LEFT,
    RIGHT,
    UP,
    DOWN
};

// 默认摄像机参数
const float YAW = -90.0f;
const float PITCH = 0.0f;
const float SPEED = 2.5f;
const float SENSITIVITY = 0.1f;
const float ZOOM = 45.0f;


class Camera {
public:
    // 摄像机属性
    glm::vec3 Position;
    glm::vec3 Front;
    glm::vec3 Up;
    glm::vec3 Right;
    glm::vec3 WorldUp;
    // 欧拉角
    float Yaw;
    float Pitch;
    // 摄像机选项
    float MovementSpeed;
    float MouseSensitivity;
    float Zoom;

    // 构造函数
    Camera(
            glm::vec3 position = glm::vec3(0.0f, 0.0f, 3.0f),
            glm::vec3 up = glm::vec3(0.0f, 1.0f, 0.0f),
            float yaw = YAW,
            float pitch = PITCH
    );

    // 返回视图矩阵
    glm::mat4 GetViewMatrix();

    // 处理键盘输入
    void ProcessKeyboard(Camera_Movement direction, float deltaTime);

    // 处理鼠标输入
    void ProcessMouseMovement(float xoffset, float yoffset, GLboolean constrainPitch = true);

private:
    // 根据摄像机的欧拉角更新摄像机的前、右和上向量
    void updateCameraVectors();
};

#endif
#include "Camera.h"

Camera::Camera(glm::vec3 position, glm::vec3 up, float yaw, float pitch) : Front(glm::vec3(0.0f, 0.0f, -1.0f)), MovementSpeed(SPEED), MouseSensitivity(SENSITIVITY), Zoom(ZOOM)
{
    Position = position;
    WorldUp = up;
    Yaw = yaw;
    Pitch = pitch;
    updateCameraVectors();
}

移动速度计算

前面计算了每帧耗时（s/frame），再乘上速度（m/s），得到每帧前进的距离（m/frame）。单位时间内，每帧前进的距离（m/frame）乘以帧数（frame）就是距离（m）。

所以无论帧率如何，单位时间内前进的距离是固定的。

void Camera::ProcessKeyboard(Camera_Movement direction, float deltaTime)
{
    float velocity = MovementSpeed * deltaTime;
    if (direction == FORWARD)
        Position += Front * velocity;
    if (direction == BACKWARD)
        Position -= Front * velocity;
    if (direction == LEFT)
        Position -= Right * velocity;
    if (direction == RIGHT)
        Position += Right * velocity;
    if (direction == UP)
        Position += WorldUp * velocity;
    if (direction == DOWN)
        Position -= WorldUp * velocity;
}

鼠标移动计算

void Camera::ProcessMouseMovement(float xoffset, float yoffset, GLboolean constrainPitch)
{
    xoffset *= MouseSensitivity;
    yoffset *= MouseSensitivity;

    Yaw   += xoffset;
    Pitch += yoffset;

    if (constrainPitch)
    {
        if (Pitch > 89.0f)
            Pitch = 89.0f;
        if (Pitch < -89.0f)
            Pitch = -89.0f;
    }

    updateCameraVectors();
}

获取视角矩阵

视角矩阵将世界坐标变换到摄像机坐标

glm::mat4 Camera::GetViewMatrix() {
    //计算视图矩阵
    return glm::lookAt(
            Position,//摄像机位置
            Position + Front,//目标位置
            Up//上方向向量
    );
}

更新摄像机向量

鼠标移动修改欧拉角值，根据欧拉角计算摄像机基向量。

摄像机基向量用于计算视角矩阵。

void Camera::updateCameraVectors() {
    //根据欧拉角重新计算摄像机基向量
    glm::vec3 front;
    front.x = cos(glm::radians(Yaw)) * cos(glm::radians(Pitch));
    front.y = sin(glm::radians(Pitch));
    front.z = sin(glm::radians(Yaw)) * cos(glm::radians(Pitch));
    Front = glm::normalize(front);
    // 同时也要重新计算右向量和上向量
    Right = glm::normalize(glm::cross(Front, WorldUp));
    Up = glm::normalize(glm::cross(Right, Front));
}
