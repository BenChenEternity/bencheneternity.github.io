两无限大平板间层流

关键假设

  1. 定常

    对于任意场中物理量 \(B\)，有 \[ \frac{\partial B}{\partial t}=0 \tag{3} \]

  2. 不可压缩

    密度为常数 \[ \rho=Constant \tag{4} \]

  3. 层流，仅 x 方向流动，y和z方向的速度无需讨论 \[ \begin{cases} u_y = 0 \\ u_z = 0 \end{cases} \tag{5} \]

  4. 无限大平板 \[ \frac{\partial u_x}{\partial z}=0 \tag{6} \]

  5. 温度变化不大 \[ \mu=Constant \tag{7} \]

  6. 质量力仅有重力

    重力方向向下 \[ \begin{cases} f_x = 0 \\ f_y = -g \\ f_z = 0 \end{cases} \tag{8} \]
