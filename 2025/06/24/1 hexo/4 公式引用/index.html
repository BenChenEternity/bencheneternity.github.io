4 公式引用

NicolasDavout 2025-06-24 13:19:11
Categories： Tags：

背景

在写文章的时候，用到很多数学公式

  • 看文章的时候，“由公式(5)”还要翻到前面去看公式(5)是什么内容，非常麻烦
  • 写文章的时候，还要翻到前面看引用公式是哪条，非常麻烦
  • 如果在前面插入一条公式，后面的公式全部都要后移一个编号
  • 公式编号更改，文中所引用的公式也要更改编号

下载

hexo_md_editor.exe

解决方案

  • 在网页的文章中提供悬浮窗，直接把鼠标移上去就能显示公式

  • 写一个软件，功能如下
    • 自动重新编号
    • 自动更新公式引用编号
    • 在文章内容编写时，提供快速复制引用能力

公式引用规则

约定一种规则，让公式能够被引用显示

在markdown中，约定如下：

这是一个行内公式的例子 <span class="formula-ref" data-formula="E = mc^2">公式(1)</span>，你可以看到它的效果。

同样，我们也可以引用一个更复杂的公式，比如麦克斯韦方程组的一条：<span class="formula-ref" data-formula="\nabla \cdot \mathbf{B} = 0">公式(2)</span>。

/*=============== 在文件 source/css/main.css 末尾添加 ===============*/

/* 公式引用文本的样式 */
.formula-ref {
  color: #21a675; /* 一个醒目的颜色，你可以自己调整 */
  border-bottom: 1px dashed #21a675; /* 添加虚线底线以示区分 */
  cursor: help; /* 鼠标悬浮时显示帮助手势 */
  font-weight: bold;
}

/* 公式提示框（Tooltip）的样式 */
.formula-tooltip {
  position: absolute; /* 关键！用于JS动态定位 */
  background-color: #333; /* 深色背景 */
  color: #fff; /* 白色文字 */
  padding: 10px 15px;
  border-radius: 5px;
  box-shadow: 0 4px 8px rgba(0,0,0,0.3);
  z-index: 100; /* 确保在最上层显示 */
  max-width: 90vw; /* 最大宽度为视口宽度的90%，适应移动端 */
  overflow-x: auto;  /* 当内容超出时，只在水平方向显示滚动条 */
  pointer-events: none; /* 避免鼠标与提示框自身发生交互 */
}

/* 确保提示框内的 KaTeX 公式颜色正确 */
.formula-tooltip .katex {
  color: #fff;
  font-size: 1.2em; /* 让提示框里的公式稍大一些，更清晰 */
}

js文件修改

//=============== 在文件 source/js/main.js 末尾添加 ===============//

function initializeFormulaTooltips() {
  // 移除旧的事件监听，防止重复绑定
  $('.formula-ref').off('mouseover mouseout');

  // 为所有 .formula-ref 元素绑定事件
  $('.formula-ref').on('mouseover', function(e) {
    // 1. 从 data-formula 属性获取 LaTeX 字符串
    const formulaString = $(this).data('formula');
    if (!formulaString) return;

    // 2. 创建 tooltip 元素
    const tooltip = $('<div class="formula-tooltip"></div>');
    
    // 3. 使用 KaTeX 渲染公式到 tooltip 中
    try {
      katex.render(formulaString, tooltip[0], {
        throwOnError: false, // 即使公式有误也不抛出错误，而是显示错误信息
        displayMode: true // 使用显示模式，让公式看起来更舒展
      });
    } catch (err) {
      tooltip.text('Error rendering formula');
      console.error(err);
    }

    // 4. 将 tooltip 添加到 body
    $('body').append(tooltip);

    // 5. 计算并设置 tooltip 的位置
    // 让它出现在引用元素的正上方
    const refPos = $(this).offset();
    const refHeight = $(this).outerHeight();
    const tooltipWidth = tooltip.outerWidth();
    const tooltipHeight = tooltip.outerHeight();

    let top = refPos.top - tooltipHeight - 10; // 向上偏移10px
    let left = refPos.left - (tooltipWidth / 2) + ($(this).outerWidth() / 2);

    // 防止 tooltip 超出屏幕左边界
    if (left < 0) {
      left = 5;
    }
    // 防止 tooltip 超出屏幕右边界
    if (left + tooltipWidth > $(window).width()) {
      left = $(window).width() - tooltipWidth - 5;
    }

    tooltip.css({
      top: top + 'px',
      left: left + 'px'
    });

  }).on('mouseout', function() {
    // 6. 鼠标移出时，移除 tooltip
    $('.formula-tooltip').remove();
  });
}
