电脑共享硬盘

NicolasDavout 2025-08-18 11:04:26
Categories： Tags：

共享硬盘

背景

来到工位之后，自己带了一台台式机，实验室提供一台台式机。两台电脑均为 windows 11。

想要让这两台电脑之间直接共享硬盘，无需使用移动硬盘中转数据。

网上的教程千篇一律，并且还有潜在的安全问题。

说明

  • windows 把网络分为两类：公用网络和专用网络，应该把以太网改为专用网络
  • 文件共享默认使用windows凭据，比如电脑B想要访问A的硬盘。此时需要在A上创建一个账号，需要账号和密码，然后在B中添加A的凭据，凭据信息包括：ip地址、用户名、密码。其中ip地址用于识别计算机，用户名和密码和A上创建的账号一模一样。
  • 为了保证安全，网络ip分配只需要容下尽量少的计算机，此处只需要2个ip地址的网段。并且全部走私有网络。在防火墙仅仅允许来自这个网段的流量。

思路

  • 网线连接电脑，直接形成物理隔离的局域网
  • 把局域网设置为专用网络，并且共享均在专用网络进行
  • 共享硬盘，并且启用用户认证
  • 添加防火墙规则，防止非局域网文件共享数据流量

实践

连通电脑

2台电脑，直接配置掩码为 30位，即 255.255.255.252，仅允许2台计算机。

避开其他内网冲突的地址，使用 192.168.0.0/30 网段，两台计算机的以太网 ip 地址分别手动配置为 192.168.0.1 和 192.168.0.2。

网关和DNS都配置为对方，因为如果不配置会导致windows识别为“未知的网络”，强制采用公用网络策略，导致没法走专用网络。

网络和 Internet 设置 - 以太网 - 编辑IP分配

IP分配：手动，IPv4 地址:192.168.0.1，IPv4掩码:255.255.255.252，网关:192.168.0.2，DNS:192.168.0.2

另外一台电脑同理

共享设置

打开两台电脑所有专用网络的共享设置

网络和 Internet 设置 - 高级网络设置 - 高级共享设置 - 专用网络下面所有的选项都打开，公用网络的全部保持默认的关闭

所以网络下打开“密码保护的共享”。

设置专用网络

把网络设置为专用网络

网络和 Internet 设置 - 以太网 - 专用网络

然后进行连通测试，是否能互相 ping 通。

如果不能 ping 通需要查看双方计算机防火墙是否开放 ICMP 回显。

控制面板Defender 防火墙 - 高级设置 - 入站规则 - 文件和打印机共享（回显请求 - ICMPv4-In）专用

该选项“已启用”修改为“是”

创建账户

两台电脑都创建本地账户

设置 - 账户 - 添加账户

无需联网登录，点击“我没有…的信息”就会进入本地账户创建

开启共享

对需要共享的硬盘执行共享，权限开启的账户为刚刚输入的账户（删除原来的everyone），打开“完全控制”权限。这样只有指定账户能访问。

添加凭据

在第一台电脑： 凭据管理器 - windows 凭据 中，添加 192.168.0.2，账号和密码同上。

添加网络虚拟硬盘

在此电脑中，点击三个点，下拉有选项“映射网络驱动器”，选择盘符之后，地址填写示例：

\\192.168.0.2\c

其中c是被共享硬盘的名字，在另外一台电脑共享设置的时候填写（默认是盘符，比如c盘共享时如果不填写名称就会默认是c）。

至此就能访问另外一台电脑的硬盘。

共享剪贴板

背景

两个电脑之间一些字符和文件需要交换，但是手动发送和接收比较麻烦。可以通过软件来实现一台电脑上复制后在另一台电脑上粘贴。

实现

使用 Ditto 剪贴板工具来实现。只需要如下操作

设置发送剪贴板内容给朋友

填入ip地址等等

在复制内容之后即可自动发送给配置好的电脑
